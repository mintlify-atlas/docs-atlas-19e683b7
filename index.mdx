Cart Management System is a C# console application that lets you browse products, manage a shopping cart, apply discount vouchers, and complete a checkout — all from a terminal interface. Built with an N-Tier architecture, it supports three interchangeable storage backends: in-memory, JSON file, and SQL Server database.

## Get started

<Steps>
  <Step title="Open in Visual Studio">
    Clone the repository and open `Cart Management.sln` in Visual Studio.
  </Step>
  <Step title="Choose a storage backend">
    In `Program.cs`, instantiate `InMemoryDataService`, `JsonFileDataService`, or `DatabaseDataService` as your `ICartDataService`.
  </Step>
  <Step title="Run the application">
    Press **F5** or click **Start**. The console menu appears automatically.
  </Step>
  <Step title="Place an order">
    Use the numbered menu to add items, apply a voucher code, and checkout.
  </Step>
</Steps>

<Note>
  Prices are shown in Philippine Peso (PHP). The default storage backend is `InMemoryDataService` — data is lost when the application exits. Switch to `JsonFileDataService` or `DatabaseDataService` for persistence.
</Note>
